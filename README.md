# knobs

## About

Display and set system tunables. See `--help` for details.

## Supported tunables

- cpu
  - online status
- cpufreq
  - min/max frequencies
  - governor
- intel_pstate
  - epb
  - epp
- drm
  - i915
    - min/max/boost frequencies
- nvml (nvidia management library)
  - graphics clock min/max frequency
  - power limit
